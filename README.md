# TwitterFake

O **TwitterFake** é uma aplicação web desenvolvida em Java com Spring Boot, inspirada nas funcionalidades básicas de uma rede social de microblogging. O projeto simula recursos semelhantes aos do Twitter/X, permitindo a criação, listagem e gerenciamento de postagens, com foco no desenvolvimento de uma API backend estruturada, organizada e conectada a banco de dados relacional.

Este projeto foi criado com o objetivo de praticar conceitos importantes do desenvolvimento backend com Java, como criação de APIs REST, persistência de dados, validação de informações, integração com banco MySQL e controle de versionamento do banco com migrations.

## Tecnologias utilizadas

- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Hibernate
- MySQL
- Flyway Migration
- Bean Validation
- Maven

## Funcionalidades

- Criação de postagens
- Listagem de postagens
- Persistência dos dados em banco MySQL
- Validação dos dados enviados
- Estruturação de API REST
- Controle de alterações no banco de dados com Flyway
